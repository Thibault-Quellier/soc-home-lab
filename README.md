# SOC Home Lab – Blue Team Portfolio

This repository showcases a complete SOC lab designed to simulate real-world enterprise security monitoring, detection, and incident response.

The goal is to demonstrate hands-on operational skills for a Junior SOC Analyst / Cybersecurity Analyst position.

---

#  Infrastructure Overview

- Wazuh SIEM (Ubuntu Server)
- Windows Server as Active Directory Domain Controller
- Windows Client joined to the domain
- Kali Linux attacker machine
- Isolated SOC lab network

---

#  Security Capabilities

- Centralized log collection
- Windows Security Auditing
- Sysmon telemetry
- Active Directory monitoring
- Attack detection and investigation
- Incident response documentation

---

#  Labs Structure

Each lab contains:
- Attack scenario
- Detection & alerts
- Investigation process
- Evidence (screenshots / logs)
- Final SOC incident report

---

#  Objective

This lab is used for:
- Professional portfolio
- SOC analyst job applications
- Continuous training & purple team practice

---

#  Repository Structure

- `/infrastructure` → Lab architecture & tools
- `/labs` → Attack & detection scenarios
- `/reports` → SOC report templates
- `/skills.md` → Technical skillset
